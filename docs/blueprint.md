# **App Name**: LeadGen Form

## Core Features:

- Form Input: Collect user data: first name, last name, gender, age, phone number, email, company name, company type, job title, and email subscription preference.
- Privacy Agreement: Require user agreement to privacy terms with a link to the full policy.
- PDF Download: Initiate a PDF download (from https://drive.google.com/file/d/1rn3plL3GEkxmgCsoRl4AXI5WlLFI4g-R/) upon form submission.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to create a sense of trust and calm.
- Background color: Light grayish-blue (#E6F3F9) to provide a soft, unobtrusive backdrop.
- Accent color: Teal (#008080) for calls to action and important links, providing a contrast that draws the eye.
- Font: 'Inter' (sans-serif) for both headings and body text to ensure clear, modern readability.
- Use simple, professional icons for form fields to aid usability without distracting from the form's purpose.
- Use a clean, single-column layout for the form to guide the user through each field in a logical order.
- Subtle loading animations upon form submission and a clear confirmation message upon successful PDF download.
